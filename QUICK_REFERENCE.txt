## ğŸ› Quick Fixes

**App won't connect to backend?**
- Verify backend terminal shows: "Application startup complete"
- Check `api_service.dart` has `baseUrl = 'http://localhost:8000'`

**Backend error on start?**
- Delete `vehicles.db` and restart
- Run `pip install -r requirements.txt`

**Port 8000 already in use?**
- Kill existing: `netstat -ano | findstr :8000` then close process
- Or change port in `uvicorn` command
